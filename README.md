# AO3 中文助手

AO3中文助手是一个Chrome浏览器插件，用于改善简体中文用户在Archive of Our Own (AO3)上的使用体验。

## 功能

- 汉化AO3常见用户界面
- 提供「翻译本章」按钮
- 提供「翻译备注」按钮
- 使用Google Translate进行即时翻译

## 支持网站

https://archiveofourown.org

## 安装方式

### Chrome Web Store

插件上线后可以直接在Chrome Web Store安装。

### 手动安装

1. 下载本仓库  
2. 打开Chrome扩展页面  
   chrome://extensions  
3. 打开 **开发者模式**  
4. 点击 **加载已解压的扩展程序**  
5. 选择插件文件夹

## 权限说明

插件只会访问以下网站：

- archiveofourown.org
- translate.googleapis.com

用于实现页面翻译功能。

本插件不会收集、存储或上传任何用户个人数据。

## 第三方服务

本插件使用Google Translate提供翻译功能。  
需要翻译的文本可能会被发送到Google的翻译服务。

Google 隐私政策：  
https://policies.google.com/privacy
