# Privacy Policy

This Chrome extension provides Simplified Chinese translations for the Archive of Our Own (AO3) website interface and content.

## Data Collection

This extension does not collect, store, or share any personally identifiable user data.

## Website Content

The extension reads text content from AO3 pages in order to display translated versions of the interface and works.  
Translation requests may be sent to Google Translate services to generate translated text.

## Data Storage

The extension does not store browsing history, personal data, or user activity.

## Third-Party Services

Text selected for translation may be sent to Google Translate services solely for the purpose of generating translations.

## Changes

This privacy policy may be updated if the extension's functionality changes in the future.

## Contact

If you have any questions about this extension, please contact the developer through the Chrome Web Store listing.
